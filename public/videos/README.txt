Signara — Video assets folder
=============================

Drop sign-language clips here using the canonical filename convention:

  hola.mp4
  ayuda.mp4
  urgente.mp4
  necesitar.mp4
  yo.mp4
  ...

The mapping is defined in src/utils/signMap.js.

If a video is missing, the AvatarPlayer falls back to a stylish branded
placeholder so the demo never breaks.
